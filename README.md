# Study of Robot Dynamics

Independent study of rigid body kinematics and dynamics for serial manipulators, built from first principles using the Product of Exponentials formulation.

## Projects
P1 — Product of Exponentials Kinematics
P2 — Jacobian Analysis and Singularities
P3 — Rigid Body Dynamics: Lagrange and Newton-Euler
P4 — Trajectory Planning with Dynamics
P5 — Vision-Based Control

## Setup
execute setup.sh to initialize a conda environment and install the dependencies.
